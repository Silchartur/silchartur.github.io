360 VBC - Plataforma Médica y Deportiva

domain Valencia Basket · Prácticas DAW · 2026

Desarrollo integral de una plataforma interna centralizada para el Departamento de Rendimiento Deportivo. Para este proyecto diseñé una arquitectura orientada a servicios (BFF) que asegura escalabilidad y separación de responsabilidades:

  • Core y API REST (FastAPI): Construcción de un motor de datos de alto rendimiento, optimizando consultas a la base de datos relacional y gestionando la lógica de negocio.
  • Frontend SSR (Django): Implementación de la capa de presentación utilizando vistas y plantillas para consumir la API de forma eficiente.
  • Seguridad Granular (RBAC): Sistema de permisos inyectados en token para segmentar dinámicamente la interfaz y las rutas de la API.
  • Analítica y Reportes: Integración de Chart.js para visualización de dashboards y automatización de PDFs.
